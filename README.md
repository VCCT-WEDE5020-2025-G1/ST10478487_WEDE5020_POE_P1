# The Kettle Café – Website

Welcome to the official website project for **The Kettle**, a cosy neighbourhood café focused on great coffee, homemade food, and community connection.

 **Overview**
This website showcases The Kettle’s menu, events, and atmosphere while providing key features such as:

- Café information & opening hours
- Full menu with coffees, cakes, and sandwiches
- Event listings (e.g., Sip & Sketch, Bake & Learn)
- Gallery and Instagram feed integration
- Contact form and location map
- Mobile-responsive design for all devices

 **Tech Stack**
- **HTML5 / CSS3 / JavaScript**
- **Framework:** Plain HTML, optionally React.js or Next.js
- **Styling:** Custom CSS (modern grid and responsive design)
- **Backend (optional):** Node.js / Express / Firebase (if needed)
- **Hosting:** GitHub Pages / Netlify / Vercel

## Site Structure (Sitemap)
1. **Home Page**
   - Header with navigation
   - Hero section with café image and welcome text
   - About section describing the café and its values
   - Clickable blocks linking to Menu and Events
   - Footer with copyright

2. **Menu Page**
   - Full menu listing coffees, pastries, and meals
   - Prices, descriptions, and images
   - Option to navigate to events or contact

3. **Events Page**
   - Event highlights with descriptions and images
   - Examples: 
     - Sip & Sketch mornings
     - Bake & Learn workshops
     - Coffee tasting & brewing demos

4. **Contact Page**
   - Contact form for inquiries
   - Newsletter subscription option
   - Embedded map showing café location

5. *Social Media*
   - Instagram and Facebook feeds integrated
   - Encourages community engagement and updates

  **Features**
- Fully *responsive design* for mobile, tablet, and desktop
- *Easy navigation* with clear sitemap
- *Event highlights* to promote community activities
- Integrated *social media* for wider reach
- *Contact options* including newsletter subscription
- High-quality images throughout the site
- Hover effects, interactive buttons, and visual enhancements for engagement

 **Purpose**
The website is designed to improve customer engagement, showcase The Kettle Café’s offerings, promote events, and make it easier for customers to find and visit the café. It also serves as a professional portfolio project demonstrating web development skills including responsive design, layout structuring, and interactive UI elements.
